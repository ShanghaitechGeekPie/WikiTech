# 学在科大

上科大的学业节奏较快，了解规则对于顺利完成学业至关重要。

## 选课指南

### 选课系统
- **登录方式**：Egate -> 教学服务 -> 选课系统。
- **阶段**：通常分为第一轮选课、第二轮选课、开学后补退选三个阶段。
  - **第一轮选课**：不分先后，人数超标时抽签。
  - **第二轮选课**：先到先得。
  - **开学后补退选**：开学后的前两周，可试听并调整。

### 学分限制
- 每学期建议学分在 15~25 之间。超过 28 学分通常需要额外申请。

## GPA 与绩点

### 换算规则

上科大采用 4.0 绩点制：
- A (4.0), A- (3.7)
- B+ (3.3), B (3.0), B- (2.7)
- C+ (2.3), C (2.0), C- (1.7)
- D (1.0), F (0)

等级和绩点的详细换算规则如下（对应的百分制分数仅供参考）。

采用等级制的课程：

| 等级 | 绩点  | 百分制（仅供参考） |
|----|-----|-----------|
| A+ | 4.0 | 95-100    |
| A  | 4.0 | 90-94     |
| A- | 3.7 | 85-89     |
| B+ | 3.3 | 80-84     |
| B  | 3.0 | 75-79     |
| B- | 2.7 | 70-74     |
| C+ | 2.3 | 67-69     |
| C  | 2   | 63-66     |
| C- | 1.7 | 60-62     |
| F  | 0   | 0-59      |

采用通过制的课程：

| 等级 | 绩点  | 百分制（仅供参考） |
|----|-----|-----------|
| P  | N/A | ≥60       |
| NP | N/A | <60       |

采用通过制的课程不计入GPA。

目前，2025级及以后入学的本科生每学期可以在限定范围内选择一门采用等级制的课程，使其采用通过制记载成绩（即不计入GPA），具体参见[关于试行本科生可选择以“通过制（P/NP）”记载课程成绩的公告（适用于2025级及以后入学的本科生）](https://oaa.shanghaitech.edu.cn/2025/1201/c4083a1117529/page.psp)。

### 重修与刷分
- 如果课程不及格 (F)，必须重修。
- 部分课程允许在成绩不理想（如 D）时选择重修以覆盖旧成绩（视具体学院规则而定）。

## 奖学金

学校设有多种奖学金以激励表现优异的同学：
- **国家奖学金**：每人 16000 元/年。
- **上海市奖学金**：每人 16000 元/年。
- **国家励志奖学金**：每人 6000 元/年，资助家庭经济困难的优秀学生。
- **校内奖学金**：
  - **院长奖学金**：本科生最高荣誉。
  - **年度优秀学生奖学金**（一/二/三等）。
  - **单项奖学金**（学术提升、社会服务、进取奖等）。

## 交流项目

### 暑期科研 (Summer Research)
大三暑假，许多同学会前往海外顶尖高校（如伯克利、麻省理工、卡内基梅隆等）进行科研训练。

### 交换生项目 (Study Abroad)
学校与多所海外高校有学期交换协议。详细信息请关注 **国际事务处 (OIA)** 的通知。

## 实验室与创新

- **大学生创新创业项目 (大创)**：鼓励本科生早期进入实验室参与科研。
- **开放实验室**：主要学院（SIST, SPST, SLST）的实验室通常对感兴趣的同学持开放态度，建议积极联系导师。
